### Регистрация и Авторизация

1. Пользователю необходимо ввести на экране авторизации свой логин и пароль от приложения и нажать на кнопку
    - Логином к приложению выступает e-mail
2. Если пользователь существует в системе - выполнить вход в личном кабинет (Кнопка - “Войти”)
3. Если пользователя не существует в системе - вывести ошибку и предложить зарегистрироваться (Кнопка “Регистрация”)
    - Пользователь переходит на экран регистрации
    - Пользователю необходимо ввести
        1. Логин (e-mail)
            - Проверять логин на соответствие шаблону (primer@primer.ru)
        2. Имя
            - Имя должно содержать не менее 3 символов
        3. Пароль
        4. Подтвердить пароль
            - Необходимо проводить проверку на соответствие 
        5. Во всех негативных сценариях выводить ошибку
    - По завершению регистрации вернуть пользователя на экран авторизации
    - Пока пользователь не прошел модерацию, он не может пользоваться функционалом приложения
        1. Клиенту отображать экран с информацией “Ожидайте проверку профиля

### Описание функционала 

#### “Верхней области”

1. В Верхнем правом углу необходимо вывести область в которой будет указываться общий баланс всех открытых счетов клиента.
    - Способ отображения общего баланса = NNNNN.nn
2. Текущая дата
3. Имя текущего пользователя 
4. Кнопка “Настройки”
    - Открывается слой “Настройки профиля”
5. Фотография пользователя
    - Если фотографии нет, выводить иконку профиля
6. Кнопка “Выход из приложения”
    - По кнопке пользователь выходит из приложения на экран “Авторизация”

#### “Область счетов”

1. Кнопка “открыть счет”
    - По кнопке клиенту предлагается открыть счет
    - Перед открытием счета (после клика по кнопке) у пользователя уточняя желание об открытие счета.
    - Счет - это всегда 10 значное число,которое начинается с цифры (4)
        - К примеру “4000000015”
2. Если у клиента несколько счетов
    - Добавить в выпадающем списке выбрать счет

#### Область “Управление счетом”

1. Указать № счета - выбранного в “области счетов”
2. Указать Баланс выбранного счета в формате NNNNN.nn
3. Кнопка “Пополнить”
    - Открывается окно пополнение счета
4. Кнопка “Перевод”
    - Открывается окно переводы с полями
        - № счета - куда совершить перевод
        - Сумма перевода
        - Кнопка “Выполнить”
5. Кнопка “Платеж”
    - Открывается окно платежа с полями
        - № счета
        - Реквизиты платежа
            1. Название платеж
            2. Название на кого платеж
            3. E-mail на кого платеж
        - Назначение платежа
        - Если есть шаблон
            1. Кнопка “Платеж по шаблону”
6. Кнопка создать шаблон
    - Открывается окно создания платежей с полями из п.5
7. Кнопка “Выписка”
    - Открывается окно с реквизитами счета
        - Имя пользователя
        - E-mail
        - № счета
        - История операций по счета
8. Кнопка “закрыть счет”
    - Открывается окно с согласием на закрытие счета
        - Если клиент согласен - закрыть счет


#### Область “История операция”

1. Необходимо указать все операции которые происходили за весь период время
2. Типы операци
    - Операции пополнение счета
    - Расходные операции по счету
        - Делить по типу
            1. Перевод
            2. Платеж
            3. Платеж по шаблону
3. Возможность отобразить все операции за определенный период времени


#### Область “Чат с поддержкой”

1. Отобразить иконку чата в нижнем правом углу
2. Отобразить кнопку “Написать в чат” рядом с иконкой
    - По клику на кнопка “Написать в чат”
        - Открывается диалоговое окно

#### Область “Настройка профиля”

1. Отобразить иконку настройки возле фотографии профиля пользователя
2. По клику отобразить слой “Настройка профиля”
    - Возможность добавить фотографию пользователя
    - Указать поля
        - Имя
        - E-mail
        - Статус
    - Кнопка “Изменить профиль”
        - Добавить возможности смены:
            1. Имя
            2. E-mail
    - Кнопка “Изменить пароль”
        - Сменить пароль пользователя
    - Кнопка “Удалить аккаунт пользователя”
    - Кнопка “Закрыть”
        - По клику закрывается настройка профиля
